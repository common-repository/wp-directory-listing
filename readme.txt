== Description ==
Easy listing creating solution in WordPress

Create and manage listing of directory items easily using WordPress with so many customizable features.

[See Demo](https://demo.pluginrox.com/wp-directory-listing/)
`
    Username: wpdl
    Password: pluginrox
`

<strong>Plugin Features</strong>

* Add/Edit/Delete Directory Items
* Custom Location
* Archive page with filters
* Complete developer friendly


=== Frequently Asked Questions ===

    = Is this plugin supports templating to customize from theme =

    Yes! It is. Anyone can just copy templates from plugin and create a folder named `wpdl` then paste which template wants to edit.

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Contact Form 7 - Responses</strong>" activate it.<br />
